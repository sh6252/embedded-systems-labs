# Session 13 - Raspberry Pi GPIO with Python

Introduction to Raspberry Pi GPIO programming using Python.

This session covers a simple digital output example with LED blinking on real Raspberry Pi hardware.

---

## Projects

- `blink.py` - Basic LED blink using GPIO17

---

## Features

- Raspberry Pi GPIO programming
- Digital output
- LED blinking
- `gpiozero` library

---

## LED Blink

Simple LED blink using GPIO17.

Features:

- LED ON/OFF every 1 second
- Console status output
