# Session 09 - Connect to WiFi 🌐

Welcome to Session 09! Here you will find two simple and clean examples that show how to connect an ESP32 to WiFi and work with external input/output.

## What is included?

- `led_with_switch/` - a switch and LED with WiFi connectivity.
- `ultrasonic_sensor/` - ultrasonic distance measurement and a network connection.

## How to use

1. Open the folder in PlatformIO or the Arduino IDE.
2. Add your WiFi details to `arduino_secrets.h` in each subfolder.
3. Upload the code to your ESP32.
4. Open the Serial Monitor and watch the messages and readings.

## Why this is cute

- both examples combine simple hardware with internet connectivity.
- it is easy to extend into a small IoT project.

## Note

Each subfolder also includes a `ReadMe.adoc` file with more details.

Enjoy! 😊
